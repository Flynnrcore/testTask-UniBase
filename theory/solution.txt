Представленный в задании код выведет в консоль:
Bad: undefined
Bad: undefined
Bad: undefined
Bad: undefined.
Ошибка связана из-за неправильной реализации замыкания цикла for, т.к. 
для объявления переменной 'i' используется var, который имеет функциональную область видимости.
Когда функция в setTimeout будет вызвана, цикл for уже завершится и значение переменной i будет равно arr.length.
Переменные, объявленные с использованием var, видны только внутри функции, в которой они были объявлены.

Для решения возникшей проблемы предлагаю два способа:
1. Обернуть setTimeout в функцию для создания функциональной видимости видимости и сразу же вызвать ее с аргументом i:

const arr = [10, 12, 15, 21];
for (var i = 0; i < arr.length; i++) {
  (function (index) {
    setTimeout(function () {
      console.log(arr[index] > 13 ? `Good: ${arr[index]}` : `Bad ${arr[index]}`);
    }, 3000);
  })(i);
}

2. Использовать актуальный способ объявления перемеменной let, который имеет блочную область видимости:

const arr = [10, 12, 15, 21];
for (let i = 0; i < arr.length; i++) {
  setTimeout(function () {
    console.log(arr[i] > 13 ? `Good: ${arr[i]}` : `Bad ${arr[i]}`);
  }, 3000);
}
